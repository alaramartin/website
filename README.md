# My Personal Website

[alaramartin.vercel.app](https://alaramartin.vercel.app)

made to learn nextjs and get better at react

## features!

-   projects filter
-   projects + blog search bar
-   using URL params in links (click on a skill in the skills section of the homepage to get automatically redirected to the projects page with the filter pre-applied)
-   silly game on 404 page
-   blog page using dynamic routing
-   random little css animations throughout (learned for the first time!)
-   mobile and cross-platform compatible
-   inverted color mode
-   metadata for good SEO
